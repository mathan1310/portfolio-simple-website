# portfolio-simple-websitePortfolio Website
This repository contains the source code for my personal portfolio website built with pure HTML and basic CSS. The site is intentionally simple and consists of just two pages:

Home: Serves as the landing page, introducing me and showcasing my work.

Contact: Provides visitors with a way to get in touch.

Features
Minimalistic Design: Focused on clarity and simplicity.

Responsive Layout: Optimized for both desktop and mobile devices.

Fast Loading: Built with lightweight HTML and CSS for quick load times.

Technologies Used
HTML5: For structuring the content.

CSS3: For styling and visual presentation.
